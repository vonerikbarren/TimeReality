## Importing Media
|- MediaPool < RightClick < New Media || Cmd + I 
   |- Need a part of a clip? 
      |- Drag to start point :: Press I 
      |- Drag to end point :: Press O (letter o not zero)


## Timeline
|- HotKeys
   |- A = Select
   |- B = Blade { Purpose: To Cut }
   |- Delete = Deleting hightlighted portion including spaces with no footage.
